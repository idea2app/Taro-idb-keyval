# Taro idb-keyval shim

A minimal shim of [idb-keyval][1] for [Taro][2] framework

## Usage

### `package.json`

```json
{
  "resolutions": {
    "idb-keyval": "npm:taro-idb-keyval"
  }
}
```

### Installation

```shell
yarn
# or
pnpm i
```
